# Kernel Execution Model

Observed Reality
        │
        ▼
Evaluation Contract
        │
        ▼
Expected Reality
        │
        ▼
Deterministic Evaluator
        │
        ▼
Evaluation Result

The Kernel performs only deterministic evaluation.

The Kernel does not authorize actions.

The Kernel does not execute transitions.

The Kernel does not contain domain knowledge.

Applications interpret Evaluation Results.
