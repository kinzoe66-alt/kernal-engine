# Kernel Glossary

## Observed Reality
The immutable facts presented to the Kernel for evaluation.

## Evaluation Contract
The declaration defining the expected reality and evaluation policy.

## Expected Reality
The conditions required to satisfy the Evaluation Contract.

## Evaluation Policy
The deterministic rules governing evaluation behavior.

## Evaluation
The comparison of Observed Reality against Expected Reality.

## Evaluation Result
The immutable result produced by the Kernel.

## Kernel
A domain-independent deterministic evaluator.

## Runtime
The orchestration layer responsible for invoking the Kernel.

## Application
A domain-specific system that interprets Evaluation Results.

## Loader
A component that transforms external representations into Kernel models.

## Domain
A specific problem space implemented by an application, not by the Kernel.
